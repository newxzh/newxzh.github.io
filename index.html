About

Zhanhang Xiong (熊展航) is a PhD student at Zhejiang University working on large language models for software engineering and industrial AI.

His research focuses on:

  • LLM-based code generation
  • Industrial control software engineering
  • PLC code generation and verification
  • Cross-language program interoperability
  • LLM agents for industrial systems
